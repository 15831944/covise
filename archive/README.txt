The files in this directory are used for building COVISE installers for UNIX
systems.

How to control what gets archived on UNIX?
==========================================

($BASEARCH is $ARCHSUFFIX with 'opt' suffix removed)


extlibs-$BASEARCH-files.txt
- list (shared) libraries that COVISE/OpenCOVER depends on
  archived into $ARCHSUFFIX/co$ARCHSUFFIX.extern_libs.tar.gz

add-sharedist-files.txt
- archived into SHARED/coshared.tar.gz

add-docdistdoxygen-files.txt
- programmer documentation generated by doxygen
  archived into SHARED/codoc.doxygen.tar.gz

add-docdisthtml-files.txt
- documentation in HTML format
  archived into SHARED/codoc.html.tar.gz

add-docdistpdf-files.txt
- documentation in PDF formats
  archived into SHARED/codoc.pdf.tar.gz

add-tutorialdist-files.txt
- .net and data files used in the COVISE tutorial
  archived into SHARED/cotutorial.maps_data.tar.gz

add-exampledist-files.txt
- .net and data files for module usage examples
  archived into SHARED/coexample.maps_data.tar.gz
