Basic Navigation

Rotate camera Use the left mouse button to rotate the camera. Click on an empty space in your room and drag the mouse while keeping the button pressed.
Move camera Use the middle mouse button to move the camera. Click on an empty space in your room and drag the mouse while keeping the button pressed.
Zoom Use the right mouse button to zoom. Click on an empty space in your room and drag the mouse forward or backward while keeping the button pressed. You can also zoom using the mouse wheel.
Select objects Simply click on an object to select it. The selected object will be highlighted in red.

Add objects / Component Library

If you move the mouse to the right edge of the window, the Component Library will appear. You can pin the widget that just appeared by clicking on . This way, it will always be visible.

There are 6 groups of objects: Room, Tables, Lights, Ceiling Pendants, Medical Equipment, Accessories Simply click on a group to display the objects in that group. You can apply filters to reduce the number of objects in the list. If you click on an object additional information will be displayed.

To add an object to the scene, double click on the item or drag it to the position you would like it to appear.

Note: If you have an object selected, the properties will be shown. In this case, simply click on "Component Library" in order to view the Component Library.

Move objects

Move object To move an object, click on it using the left mouse button and start dragging. You can also move a selected obejct using the arrow keys.
Rotate object To rotate an object, click on it using the right mouse button and move the mouse. The point you clicked on, will be rotated towards the mouse pointer.
Mount / Unmount By default, new objects are mounted to the room. You can double click on them in order to unmount them. When unmounted, you can move them freely. Move them close to a wall or column and double click again to mount them on the nearby surface.
Cardanics You can move cardanic parts by clicking on them using the left mouse button and moving the mouse. This way, you will only rotate the part you clicked on. When using the right mouse button, you will move the entire arm.

Edit objects

Many objects have attributes that can be modified. To change these properties, select the object and move the mouse to the right edge of the window. If you want tp change the properties of the room, make sure no item is selected and click "Properties" at the top of the appearing widget.

Shortcuts

DEL Deletes the selected object.
P Switches to first-person view if a human is selected.
