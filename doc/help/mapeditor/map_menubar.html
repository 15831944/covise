(Map Editor) Menu Bar

 

The Menu Bar provides functions to
 

  • control a session using map files (File/Session) via File
  • execute a map (either unconditionally or in case of changes) via Execution

    •  
  • initiate and control distributed and collaborative working (see CollaborativeWorking for basic concept) via the options of CSCW  (Computer Supported Collaborative Work) and MasterControl

    • Note:
    (File, Execution, and) CSCW (in black): can be selected by Master only
                   MasterControl  (in grey):  only option a Slave can select
  • get Help  (online help, or detailed info via User's Guide)


You can use shortcuts for the operations of the Menu Bar from the Tool Bar.
