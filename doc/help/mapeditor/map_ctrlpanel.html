Control Panel (Mapeditor index card)

The Control Panel is created when COVISE starts; switch to this index card by clicking on the corresponding entry. Its purpose is the collection of graphical interactors which typically represent often used and changed parameters.

Parameters are loaded as parameter interactors (see below) to the Control Panel

  • by default if requested by a specific module or
  • explicitly if you map a parameter definition from page 1 of  Module Information window to a parameter interactor in the Control Panel via the button at the left of the parameter name (click again if you want to remove it)

 
Map Editor - Control Panel index card

Panel contains examples for scalar, vector, boolean, and choice parameters  (DomainSurface) and a scalar parameter (Colors)
 

Mapping of parameters from Module Setup to Control Panel allows you to close the Module Information window if not needed otherwise.

Notes:

If you try the right mouse button for other parameters than scalar, slider, or choice  you get: There is no appearance menu for this parameter!
 

If you select spinbox in the appearance menu for a a float scalar, you get: Paneltype not supported for float values.
 

The MapEditor suggests you a delta value, but normally you will have to adjust it depending on your data.

For Collaborative Working only - please observe the following restrictions for scalar/slider parameters if their appearance type has been changed to player/sequencer:
    • a player/sequencer will be stopped if you transfer control to the slave
    • the "delta value" will not be transferred to the slave; so if you don't like the default value you have to explicitly set delta at the slave again

Browser parameter: If you have selected modules before and not used predefined maps, the browser window has already come up automatically with the module.

The name of the module is copied to the Control Panel with the first parameter interactor. Together with the modul name you get three additional icons:

  • Exec (  icon) to execute the map starting with this module

    • Note:
    Same function provided by 
    Exec from 'Further Actions' menu (Working with Modules)
    Execute from the Module Information window 
  • Info (  resp.  icon) to get the Module Information window (if you like to see Module Information and Control Panel in parallel)
  • Collapse/Expand parameter window for this module (  resp.  icon) to show parameter name(s) only or to show parameter(s) plus interactor(s)
The bottom line of the Control Panel provides three additional functions:
  • Execute All

    • Execute the whole map
  • Execute OnChange sets OnChange in the menu item Execution - initiates execution after every single parameter change.
  • Help (see this window)
