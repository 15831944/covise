Map Converter

If you use Load Network to load a map that has been stored with a COVISE Release older than the one you are currently using, you might encounter problems because meanwhile one or more modules of your map have changed (new ports, parameters, rearrangement of functions etc.).
In this case please use the MapConverter program to get an up to date version of your map, and save the updated map for future use.
