Module Ports

Only ports that allow the same data type to be transferred can be connected
(see ConnectingModules). So you have to know the type of the ports.
 
 
Show the data type(s) allowed for a port by clicking on the port with the right mouse button:
You get the the allowed data type (or list of allowed data types) in a  message window attached to the icon (Message area not used for this purpose any longer).

The entry has the following structure:

  • port name (explanation)

    •       data type
    or
  • port name (explanation)

    •      list of data types 
    (see example at the left)

You can get detailed info about actual data types used in your map after execution if you

  • press CTRL and click on the port with the left mouse button or
  • select the corresponding data object in the DataObjects selection list .
For a list of possible data object types see COVISE Objects (DataObjectList).
