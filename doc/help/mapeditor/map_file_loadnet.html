Load Network

By Load Network you select where you get your map from (Session Open) or store your map to (Session Save).
For selection you use the MainBrowser (Open):
 
 

 

Use e. g. Path to select a group of directories and select then from Directories and Files,  or specify directly the file via Filename, or use Filter to restrict your selection to files containing a net (map).

 
In case of Session Open you choose Apply if you want to use the map from the selected file or choose Cancel otherwise.
Choosing Apply you
  • clear the working area as if New is chosen
  • load the previously stored map together with the current parameter settings
     
    Note:
    If you get error messages in connection with your map, one or more modules of your map may have been changed since the time you stored the map - use the MapConverter program to get again an up to date map!
Now you can start your session via Execute (see Execution in the Menu Bar or   in the Tool Bar)
 
In case of Session Save you choose Apply if you have selected the file you want or choose Cancel otherwise.
Choosing Apply you store the map to the selected file.
