Connecting Modules

Only ports that allow the same data type to be transferred can be connected
(see ModulePorts).
 

You can

  • connect 2 ports by clicking with the left mouse button on the output port of a module and then on the appropriate input port of the module to be connected
  • highlight a connection line by placing the cursor on the line
  • disconnect ports by clicking on the connection line
Note (easy connect):
To establish proper connections w/o looking into the detailed descriptions of the ports
  • Click with the left mouse button on the input port of the module: The output ports of modules offering the appropriate data type are highlighted (red)
  • Click on the proper highlighted port with the left mouse button to establish the connection.

After you have established all necessary connections, you can execute your map; see

  • Execution > Execute in the Menu Bar
  •   in the Tool Bar
  • 'Execute all' from the bottom line of the ControlPanel  (index card)
