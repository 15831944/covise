Map Editor

The MapEditor is the main interface for the COVISE user

You can use the MapEditor like a toolkit

  • to build a  map or 'pipeline', i. e. a dataflow network for postprocessing and visualization of simulation data
  • to execute this map
  • to show the results
    • in the IV Renderer (output area) on the desktop
    • with COVER (VR Renderer) e.g. in the CAVE
  • to modify the results by changing parameters in the ControlPanel
  • to look at the COVISE Objects (data structures) involved
The MapEditor window consists out of 4 main parts (see figure below)
  • Menu Bar
  • Tool Bar
  • Index Card according to the current function of the Map Editor
    • Visual Programming- default: to build a map
    • COVISE Objects - to look at the data structures involved
    • Control Panel - to modify parameters
  • Message Area

    • (Info Messages)
For sessions with 2 or more partners a Chat Line (in connection with the Message Area) is added (see also CSCW)
