File (Session Control)

 

A file containing a list of modules and information about the connections is called a Map.

The option File (Session Control) allows you to

  • begin a new session  - build a map to be stored in a new file (New)
  • start a predefined session - open a file containing an existing map (Open)
  • store the current session - save the map to a file (Save)
  • quit the current session and leave COVISE (Quit)
If you select New, COVISE will clear the working area, the Control Panel, and all Data Objects. Added hosts will remain in the system.
You can generate a new map by:
    • selecting modules using the ModuleBrowser
    • connecting modules (module ports), see

      •    module icon layout (ModuleIcon)
         connecting ports (ModulePorts and ConnectModule)
         working with modules (Working with Modules)
If you call Open/Save, the file browser LoadNetwork requests the location where to fetch or store the map.

In case of Open

    • the working area is cleared as if New is chosen and
    • the previously stored map is loaded together with the current parameter settings
If you select Quit, a special logout window appears. If you type 'yes' you quit COVISE and all participating processes on all participating hosts are terminated.
