3D Input Device

     
  • Sensor and 1-3 buttons

    •  
  • Sensor records position and orientation of the hand

    •  
  • Distant objects, e. g. entries in the 3D menu, can be reached by  a virtual laser beam
  • One button is used to indicate a selection, the others are used for  additional modes of navigation

    •
