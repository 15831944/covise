Headtracking

 
 
  • Sensor, mounted at the user's glasses, records the user's motion

    •  
  • Position and orientation of sensor is used to compute the view

    •  
  • Users can move within the Virtual Reality environment without restrictions

    •  
  • Only the user wearing the sensor has the correct view.

    • Additional users should try to stay close to the one with the sensor and to look in the same direction
     
  • If the scene is moving, too...

    • ... headtracking will become less accurate
