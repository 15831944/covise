How to start COVER
 
 
  •  3D Viewer


    •           –   for formats supported by Performer  (dxf, pfb, obj, iv, flt)

              –   In addition: vrml with sound and interaction

              –   to start: cover <filename>
     
     
     

  •  COVISE Renderer Module
      •  
                 –   Category Renderer, module COVER or COVER_VRML
         –  can be connected to all modules generating graphic primitives
              (lines, points, polygons, tristrips with colors, normals, textures)

         –   COVER_VRML contains in additon the vrml-loader
