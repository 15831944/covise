(Renderer) Tool Bar and Wheels

Notes:
Tool Bar and Wheels (plus Menu Bar ind Information Area) will disappear if you set 'Decoration off' in the Viewer Popup Menu (menu Covise Renderer) 
It may be necessary to resize the Renderer window in order to see the complete Tool Bar including the projection mode icon.

With the Tool Bar you can perform the following functions (see icons below):
 
  • select geometry options for manipulation and editing

    • (PickMode)
     
  • move your scene using the mouse

    • (ViewMode)
     
  • activate headtracking

    • currently not supported
     
  • ask for help: not implemented

    • general help: IV Renderer
    specific help for toolbar / wheels see this window
     
  • define/restore a default camera 

    • position/orientation 
    (HomePosition / SetHomePosition)
     
  • perform specific changes to your view

    • (ViewAll and SeekMode)
     
  • change the ProjectionMode

In addition to the mouse operations in view mode you can move your scene via three thumb wheels.
