Material Editor

The Material Editor is used for customizing objects by interactively changing values for
    • ambient, diffuse, specular, and emissive color
    • transparency
    • shininess
Note:
Before using the Material Editor, select a geometry object either by
    clicking in the GeometryObjects selection list or
    clicking on the geometry object itself (in PickMode!)

For using the editor look at the Material Editor popup together with the description below.
The Material Editor provides the Edit menu (Material List / Continuous / Manual / Copy / Paste / Help), the Control field and the Sliders (Color / Shininess / Transparency):
 

The slider values have the following meaning:

  • Ambient color:

    • reflected color of an object in response to the ambient lighting in the scene
    default [0.2 0.2 0.2]
  • Diffuse color:

    • object's base color (specified in the color array of the renderer geometry input data objects)
    default RGB [1.0 1.0 1.0] = white
  • Specular color:

    • color on highly reflecting parts = color of the incident light
  • Emissive color:

    • color of light caused by self illuminating object;
    default [0.0 0.0 0.0], i. e object is emitting no light
Note:
For all kinds of color above you can click to get a full  ColorEditor.
  • Shininess:

    • The degree of shininess is for achieving e. g. metallic effects;
    default 0.0 (diffuse surface), 1.0 for a highly polished surface
  • Transparency:

    • If you have two objects, one covered by the other, the transparency level means
     
               0.0 no transparency you see the covering object only
    0.0< transparency <1.0 limited transparency you see both objects
                 1.0 full transparency you see the covered object only
Note:
Check color and shininess looking at the sphere in the control field, check transparency looking how much of the squares you see shining thru the sphere.

Use the option 'manual' from the Edit menu to prevent changes beeing reflected immediately
