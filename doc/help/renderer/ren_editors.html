Editors / Interactor Options

 
 
As Editors and for Transforming Objects you can use the following tools (see above):
  • Material Editor...

    • Use this editor for customizing objects by interactively changing values for
    • ambient, diffuse, specular, and emissive color
    • transparency
    • shininess
    and immediately seeing the effects of these changes
    For using the editor look at the Material Editor popup together with the description.
     
  • Color Editor...

    • This editor lets you interactively change the color properties of an object.
    The same editor is called
    • to edit a light source (see Lights)
    • to edit the background color of the viewer area (see Viewing)
    • from the Material Editor
    For using the editor look at the corresponding ... Color popup (e. g. Diffuse Color) together with the  description.
     
  • Object Transform...

    • This editor provides transform sliders
    • to change the position of objects in respect to each other
    • to scale an object to make it appear larger or smaller on the screen
    For use look at the Transform Editor popup together with the description.
Note:
Before using one of the three editors above, select a geometry object either by
    clicking in the GeometryObjects selection list or 
    clicking on the geometry object itself (in PickMode! ) 
     
  • Parts... COVISE PartEditor

    • For using the editor look at the COVISE Part Editor popup together with the description.
The last 2 items (Snap handle to axis / Free handle motion) set options for an Interactor
