Lights

 
With Lights you change the appearance of an object by changing its illumination.
You can create, edit, and remove light sources in the viewer area.
Use the following options (see above):
  • Lightmodel BASE_COLOR (toggle button)

    • Each object has its own base color also called diffuse color. (If no color is specified, objects are colored white over all faces or vertices.) If BASE_COLOR is
    • enabled (default): Objects are rendered by taking in account their diffuse color only (no shading)
    • disabled: Objects are PHONG shaded. The PHONG lighting model takes into account all light sources in the scene and the object's surface orientation with respect to the lights to generate a realistic smooth shaded 3D appearance of an object.

      •  
  • Create Dir(ectional) Light

    • This creates a light which illuminates uniformly along a particular direction.
  • Create Point Light

    • A point light, like a star, radiates light equally in all directions from a given location in 3D space
  • Create Spot Light

    • A spot light illuminates from a point in space along a primary direction. Its illumination is a cone of light diverging from the light's position (hardware dependent: if not supported, spotlight appears as point light)

    The 3 options above

    • create the light source as specified
    • place the light icon at the default position in the scene
    • create a new lights entry (Directional/Point/Spot) in the Lights menu
    These new lights can be
    • edited interactively with the mouse after turning on the Light Edit mode (see option Viewing)
    • modified by a submenu of the new lights entry which offers
        • On/Off
        Icon (show/hide)
        Edit Color (gives you access to a ColorEditor for this type of light)
        Remove (new lights entry)
    (see figure above for an example of a  new directional light source)
     
  • Ambient Lighting

    • This is used for the PHONG lighting mode. It lets you edit the ambient lighting in the scene via a ColorEditor for this type of light.
     
  • Turn all on/off

    • turns all currently defined lights on/off
     
  • Show/Hide all Icons

    • shows/hides the icons of all currently defined lights
     
  • Headlights

    • This is the default light. The icon for the headlight is turned off by default.
    Headlights offers a submenu with
        On/Off
        Edit (provides a special Headlight Editor)
    With the Headlight Editor you can
    • change the intensity of the light by moving the 'Inten' slider
    • adjust the angle at which the light shines by clicking on the directional arrow and changing its position in the window
    • edit the color via ColorEditor
    Note:
    If you create new lights, new entries will appear below Headlights.
