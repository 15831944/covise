Annotation

You can use the Annotation function to add a description to the Renderer image like "isosurface" shown in the picture below.
 
 

Switch to pick mode and click on the detail you want to explain:

You can now (using a popup with "apply")

  • add a new Annotation
  • edit an existing Annotation
  • delete an Annotation
Notes:
    • Annotations can be saved as part of a map.
    • Annotations are 'static': an annotation attached to an isosurface will not move if the isovalue is changed!
