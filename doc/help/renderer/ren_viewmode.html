View Mode

You select View Mode by the following icon
 

View Mode and PickMode are the two fundamental interaction modes of the mouse in the Renderer.

Note:
You can also switch between pick mode and view mode via
    • option Viewing > Pick/Edit from the Menu Bar 
    • option Viewing of the Viewer Popup Menu  (right mouse button on viewer area)

View Mode

  • is indicated by an active view mode button in the tool bar
  • changes cursor to hand symbol
  • transforms mouse interactions in viewpoint changes (moves object or camera)
In view mode you
  • rotate the whole scene using the left mouse button:

    • Move the mouse pointer over the geometry, press the button, and move the mouse in the direction you want to rotate to.
    Note:
    You can 'show point of rotation axes' using the corresponding option of the  PreferenceSheet.
    If you stop moving the mouse first and then release the button, rotation stops immediately.
    If you keep moving the mouse while you release the button, the scene will spin permanently until you click on the viewer area.
    Note:
    'spin animation' must be enabled in the PreferenceSheet!
  • pan the scene by keeping the middle mouse button pressed
  • zoom the viewpoint by keeping the left and middle mouse button pressed:

    • move mouse to bottom - zoom into the scene
    move mouse to top - zoom out
Note:
Similar to the mouse operations in view mode you can change the view of the object(s) in the viewer area by three thumb wheels. For related tools see also Manipulators and the Transform Editor (see ObjectTransform)
