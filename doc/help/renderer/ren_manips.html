Manipulators

 
Manipulators (see menu above) are used for direct manipulation of a certain object, like the editors.
This way of manipulation is more direct than using the editors, but less precise than using the sliders.

To attach a manipulator to an object

  • select an object (click in the Geometry Objects list or switch to pick mode and click on the object itself)
  • select a manipulator type in the Manips menu
or
  • switch to pick mode
  • select a manipulator type in the Manips menu
  • click on the desired object
The manipulator appears and surrounds the object.

You can use different types of manipulators to transform, rotate, scale, and move an object in the viewer area:

  • Trackball

    • This manipulator is for rotating and scaling an object. It appears as a transparent sphere around the selected object. Use the SHIFT and ALT keys together with the left mouse button to achieve specific effects
  • Handle Box

    • This inserts a transparent cube into the scene that allows you to scale and translate the object by moving the mouse in various ways. Use the SHIFT and ALT keys together with the left mouse button to achieve specific effects
  • Jack

    • The object can be zoomed and rotated
  • Centerball

    • You can move the center point of rotation for an object. Afterwards the object can be rotated around the new rotation point.
  • Transform Box

    • This manipulator is for transforming the selected object.
  • Tab Box

    • This lets you scale an object by doing a click-drag-release motion with the mouse after clicking on the green control points.
  • None

    • This is the default. If you pick an object, no manipulator is attached.
  • Replace

    • This option allows you to work with more than one manipulator at a time: If a different manipulator is selected from the list, you can decide whether the currently active manipulator is either replaced or stays attached.
