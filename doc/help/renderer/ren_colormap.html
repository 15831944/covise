Color Maps

COVISE provides the modules Colors and ColorEdit to map scalar data to colors.

The Colormaps box in the information area of the Renderer contains the data objects that are colored.
 
 

To view / hide the color map used for an object

  • click on the data object to show the corresponding color map in the viewer area
  • click on the data object again to remove the color map
To adapt the descriptive info use module parameters, e. g.
  • change 'color' to 'pressure' or 'velocity'
  • change default min and max values to actual values
To modify the color map itself you have to use specific modules:
  • The module Colors  allows you to select predefined maps (Standard, Star, ITSM, or Rainbow)
  • The module ColorEdit allows you to edit (and save) color maps. The module ColorEdit contains a ColorMap Editor as input parameter.

    • For using the ColorMap Editor look at
    • the ColorMapEditor Panel together with a short description (see ColorMapEditor)
    • the documentation for the module ColorEdit including a detailed description of the editor (see Online Module Description List, ColorEdit)
