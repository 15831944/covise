Viewing

 
The Viewing menu item provides the following functions (see above)
  • Pick/Edit : switch between Pick Mode and View Mode (Edit mode: old terminology)

    • Note:
    Same function available via 
      • pick mode and view mode icons (Toolbar, PickMode, ViewMode) 
      • option viewing of the Viewer Popup Menu (right mouse button on viewer area)

        •  
  • Light Edit Mode: Enabling this mode lets you interactively edit the currently visible light sources in the scene by using the mouse (see menu item Lights > Create...)
  • View Selection: adjust size and position of the object so that it fits exactly in the viewer area (orientation remains unchanged)

    • Note:
    View Selection can only be selected if an object has been selected: switch to pick mode and click on the object in the viewer area, or click on the name of the object in the Geometry Objects list before you try to use View Selection! 
  • Fog: simulates various atmospheric effects such as fog, haze, pollution, and smoke which are grouped under the term fog

    •  
  • Antialiasing: eliminates or reduces jagged lines and makes objects drawn on the screen look smoother (see also: Glossary).

    • Warning: Enabling Antialiasing reduces drawing speed!
     
  • Modify transparency by selecting

    • Screen Door Transparency:
    This is the default and the only supported mode on some machines. It uses GL patterns for achieving the transparency effect.
    Blended Transparency:
    uses GL alpha blending
    Delayed Blended Transparency:
    uses GL alpha blending; opaque objects are rendered first, then transparent objects
    Sorted Blended Transparency:
    uses GL alpha blending; opaque objects are rendered first, then transparent objects; additionally objects are sorted by their distance from the camera and are drawn from back to the front
    (for more information about transparency refer to the description of the MaterialEditor)
     
  • Edit Background Color: change background color of the Renderer window (default: black) via the Background Color popup (see ColorEditor)

    •  
  • Hide Coordinate Axes: toggles between hiding and showing the coordinate axes; default is on.

    •  
  • Clipping Plane: cut off part of your geometry object(s) by a plane; via the Clipping Plane pop up you
      • specify a point on the plane and the normal
      enable the plane via Apply
      switch it on/off via Clipping Plane
    Note:
    Similar functions are available via
    CutGeometry  (Module) 1 permanent plane with variable normal and a variable point on the plane (to be supplied as module parameters)
    Clipping Planes (PreferenceSheet) 2 switchable clipping planes with fixed normals, distance varied by a wheel
