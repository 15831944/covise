Object Transform

Object Transform provides transform sliders
    • to change the position of objects in respect to each other
    • to scale an object to make it appear larger or smaller on the screen
Notes:
Before using this editor, select a geometry object either by
    clicking in the GeometryObjects selection list or 
    clicking on the geometry object itself (in PickMode!)
Remember that the changes only affect the currently selected object in a scene!

For use look at the Transform Editor popup together with the detailed description below.

In the Transform Editor window you select one of the five submenus

  • TRANSLATIONS

    • changes the position of an object in the scene
  • SCALES

    • changes the size of an object
  • ROTATIONS

    • changes the orientation of an object in the scene
  • SCALE ORIENTATION

    • changes the orientation for scale operations
  • CENTER

    • changes the center around which rotation takes place
Each submenu has three sliders to perform the desired operation (x-, y-, and z-direction); in addition, you can choose out of three layout styles for your slider widget.
