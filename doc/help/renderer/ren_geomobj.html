Geometry Objects

The GeometryObjects list box in the information area of the Renderer contains all geometry objects displayed in the viewer area below.
 

You select a geometry object (indicated by a red wireframe bounding box) in one of two ways
  • using the GeometryObjects list box:

    • click on the name of the geometry object in question with the left mouse button (works in pick mode and in view mode)
  • using the viewer area:

    • click on the corresponding image with the left mouse button
    • select one geometry object by clicking on it
    • select multiple objects by holding down the shift key while selecting
    • remove selection of an object by selecting it again
    • remove selection of all objects by clicking on any free part of the viewer area
    Note:
    You must be in Pick Mode to select an object in the viewer area! 
You manipulate a geometry object after selection with
  • Editors
  • Manipulators
  • mouse operations in ViewMode
  • thumb wheels

Since these geometry objects are a subset of the data objects listed in the DataObjects selection list of the MapEditor, you get the data type of a geometry object just by clicking on the corresponding entry in this list box:
A window pops up which shows the data type (plus other info as described in DataObjectList)
