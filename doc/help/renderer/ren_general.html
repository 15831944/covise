Renderer

(Open Inventor Renderer - IV Renderer - Desktop Renderer)

The Renderer visualizes the output data.

It consists out of 3 sections (see figure below):

  • Menu Bar
  • Information Area (Colormaps, Geometry Objects, and Collaboration Status)
  • Viewer Area plus 'Decoration' (Tool Bar and 3 Wheels)

You have 4 possibilities to select functions/options

  • Menu Bar
  • Tool Bar

    • (right of the viewer area)
  • 3 Wheels

    • (bottom corners of the viewer area )
  • Viewer Popup Menu

    • (via clicking with the right mouse button on the viewer area)


A special feature for collaborative working is the Telepointer
 

The information area provides

  • text fields for collaboration status (see option Sync in the menu bar)

    •  
  • the ColorMaps list box containing all colormaps used in the pipeline

    •  
  • the  Geometry Objects list box containing all the geometry objects currently displayed

    •  
  • a Timestep Sequencer

    •  

     
     
     

    Notes:
    If you need a larger viewer area you can "set decoration off" (Viewer Popup Menu >> Decoration), i.e. show viewer area only, w/o menu bar, information area, tool bar, and wheels.
    A Timestep Sequencer is only generated if timesteps are used in the map, e. g. in case of an animated particle trace ( therefore not shown in the figure above).
