File

 
The File option provides the following functions(see above):
  • Save

    • saves, via the File Dialog_popup, the current objects in Inventor3D = VRML1.0 format  to a specified file. Programs reading this format can load this objects and allow further usage and postprocessing of the 3D data.
    Save remembers the file you specified; following Save operations use the same target file.
     
  • Save as...

    • saves, via the the File Dialog_popup, the whole scene to a specified file.
    Use Save as... to store to different target files.
     
  • Snap

    • takes a snapshot of the viewer area.
    Note:
    Format of the snapshot files is .tiff (changed with Rel. 5.2.3). Creates an error dialog if offscreen rendering is not possible due to low colordepth. Offscreen rendering requires at least a 24bit true color. Same applies to Snap all.
  • Snap all

    • takes a sequence of snapshots of the viewer area.
    Note:
    Snap all can only be selected if the map contains timesteps, e. g. if an animated particle trace is generated. Format of snapshot files see Snap.

    You can use these snapshots to generate a simple movie (see example for SGI).

  • Copy View

    • copies the currently selected object to a buffer from which other programs like IRIS Showcase can paste the 3D object into their application.
    Notes:
    Copy View can only be selected if an object has been selected: switch to pick mode and click on the object in the viewer area, or click on the name of the object in the Geometry Objects list before you try to use Copy View! 
    You can also use Copy View in the Viewer Popup Menu  (menu COVISE Renderer, option 'Functions', available by clicking with the right mouse button on the viewer area) 
  • Print

    • saves, via the Scene Viewer Printing popup, the currently visible scene in a postscript file or sends the postscript output directly to a printer. The available printers are listed, the output qualitiy and print size in inches can be adjusted, and the page format can be chosen between landscape or portrait.
     
  • Read/Save Camera Env...

    • restores/saves, via the File Dialog_popup, the camera environment (coordinates, orientation, viewing angle)
