Pick Mode

You select Pick Mode by the following icon
 

Pick Mode and ViewMode are the two fundamental interaction modes of the mouse in the Renderer.

Note:
You can also switch between pick mode and view mode via
    • option Viewing > Pick/Edit from the Menu Bar 
    • option Viewing of the Viewer Popup Menu (right mouse button on viewer area)

Pick Mode

  • is indicated by an active button in the tool bar
  • changes the cursor to an arrow
  • allows to select geometry objects for further manipulation/editing

    • (includes objects in a bounding box)
In pick mode you
  • select a geometry object by clicking with the left mouse button on it:

    • The selection is indicated by a red wireframed bounding box around that object.
  • select multiple objects by holding down the SHIFT key while selecting
  • remove the selection of an object by selecting it again
  • remove the selection of all objects by clicking on any free part of the viewer area
Note:
You can select a geometry object (in pick mode and and in view mode) by clicking on its name in the GeometryObjects list box, too.
