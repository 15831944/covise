Viewer Popup Menu

If you click with the right mouse button on the viewer area a window with the heading Covise Renderer pops up that provides additional functions (options) that can be selected with the left mouse button (see below).
 

These functions/options can be grouped into six classes:

  • General functions

    • (similar functions available in ToolBar or MenuBar)
    • Home / Set Home (same as Home Position / Set Home Position icons in tool bar)
    • View All (same as ViewAll icon in tool bar)
    • Seek (same as SeekMode icon in tool bar)
    • Copy View (same as option File > Copy View in menu bar)

      •  
  • Draw Style options

    • (see DrawStyle)
     
  • Viewer area options
      •  
    • Viewing: switch view/pick mode (description see ViewMode and PickMode)

      • Note:
      Same function available via
      option Viewing > Pick/Edit (Menu Bar)
      view mode and pick mode icons (ToolBar)
    • Decoration: set decoration off, i.e. show viewer area only

      • (w/o menu bar, information area, tool bar, and wheels)
    • Headlight: switch headlight on/off

      • (same as in option Lights > Headlight in menu bar)

  • Preferences...

    • set defaults in the Covise Renderer PreferenceSheet
     
  • Annotation functions

    • (see Annotation)
     
  • Master Request

    • in slave mode only - request to become master, see Mapeditor > Menu Bar > MasterControl
