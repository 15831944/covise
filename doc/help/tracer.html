Module Reference Guide

Tracer:
Tracer
performs particle tracing within vector fields inside any kind of grids. It can either show Floating Particles or connect these to Streamlines, Streaklines, and Pathlines, which are different for transient data.

TracerComp
performs particle tracing within vector fields inside any kind of grids,. It can either show Floating Particles or connect these to Streamlines, Streaklines, and Pathlines, which are different for transient data. The key difference with respect to the normal Tracer module is that TracerComp relieves you of the burden of connecting the output with Colors, Collect; and Sphere modules. You may directly connect the first output port to the Renderer.
