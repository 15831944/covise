06/18/01: Tutorial in Latex and online in HTML
07/03/01: change contents of make all and make Tutorial

A Latex version of the Tutorial is provided in .../doc/Tutorial/..., the corresponding chapters of the online version (.html) can be generated in .../doc/online/Preface,  .../doc/online/StartCOVISE, ...., .../doc/online/AdvancedTopics.
The Latex version is based on the latest available .doc version; this .doc version and the .pdf version in doc/tutorial/... are untouched. (Capital "T" = new version, "t" = old version)

Remarks:

The Latex version is an update of the .doc version from snapshot 10 to Release 1. Most images and corresponding text parts had to be updated (maps, renderer output, Mapeditor menu items). In general, some other modifications have been necessary to get an acceptable html.

The Makefile provides the following options
 
all:  generate  .dvi  .html .pdf and  .ps
Tutorial generate .dvi and .pdf 
Tutorial.dvi: generate .dvi
Tutorial.ps: generate the .ps version
Tutorial.pdf:  generate the .pdf version
Tutorialhtml: generate the .html (online) version

and the reverse operations
 
clean_Tutorial:  remove .dvi .ps .pdf .toc .aux
clean_Tutorialhtml: remove online version (.html)

The following start points into the online version of the Tutorial are provided; normally start_Preface should be used as entry and this should be inserted in the help menu of the MapEditor (Online Help uses start_Preface and start_Advanced, if applicable).

start points  (in .../doc/online/start/...)
 
start point toc calls
start_Preface tocPreface Preface 
- entry to Tutorial (1st chapter)
start_StartCOV tocStartCOV Starting COVISE
start_UseRenderer tocUseRenderer Using the OpenInventor Renderer
start_ModWork tocModWork Working with Modules
start_3DData toc3DData Analysis of 3D Data
start_Advanced tocAdvanced Advanced Topics

Table of contents (list of chapters) of the Tutorial in Tutorialtoc
 

The Online Help has been updated to branch to the online (.html version) of the Tutorial, and the help menu in MapEditor uses start_Preface as well.
