You can use Meeting Room chat to send typed messages to everyone els in the meeting. You can also send
private chat messages to selected participants. Use chat if audio is not available or if you need to ask a
question and do not want to disturb the audio portion of the meeting. You can also use chat to request
permissions from the Moderator.

\vspace{0.5cm}
{\bf Sending a Meeting Room Chat Message to All Participants}
\vspace{0.5cm}

\begin{enumerate}
\item Enter your message in the text-entry area at the bottom of the Meeting Room.
\item Click Send. All meeting participants see the message in the chat-transcript area.
\end{enumerate}

\vspace{0.5cm}
{\bf Sending a Private Chat Message to Selected Participants}
\vspace{0.5cm}

\begin{enumerate}
\item If you are sending a message to one participant only, double-click on the name in the Participant
List (Connect List) and go to step 2. If you are sending a message to more than one participant, press CTRL and select
the appropriate names. Then, choose Meeting - People - New Chat and enter information about the chat
meeting.
\item Enter your message and click Send. The message appears in the selected participant's private chat
window.
\end{enumerate}
