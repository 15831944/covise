Project: CaveVOX
Authors: Jurgen Schulze, Devon Penney, Alexander Rice, Mike Shim

Description:

Volume rendering application for real-time environments. 
Has specific features for confocal imaging and visible human data.

The application expects the following variables to be set in COVERConfig:

VOLUME_DIRECTORY <path_to_volume_files>
RESOURCES_DIRECTORY <path_to_widget_textures_etc>
